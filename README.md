### automobile sales dashboard

developed an interactive **dash** dashboard using **plotly** as part of a **coursera** project to visualize yearly and recession-period automobile sales.

### purpose
this dashboard provides an interactive tool to explore automobile sales trends, economic factors, and advertising during normal and recession periods. it allows comparison of vehicle sales, advertising spend, and the effect of economic indicators such as unemployment and consumer confidence over time.

#### run instructions

1. install dependencies: 
   ```bash
   pip3.8 install setuptools
   python3.8 -m pip install packaging
   python3.8 -m pip install pandas dash
   pip install more-itertools
   
2. run the python file:

python3.8 DV0101EN-Final-Assign-Part-2-Questions.py


3. observe the port number shown in the terminal.

4. click Launch Application from the side menu bar, provide the port number, and click OK.
